# QR Code Generator

A Python application for generating QR codes with PyInstaller build support.

## Files

- `qr_generator.py` - Main QR code generator application
- `generate_icon.py` - Script to generate application icons
- `QR Code Generator.spec` - PyInstaller specification file
- `build_macos.sh` - Build script for macOS
- `build_windows.bat` - Build script for Windows
- `BUILD.md` - Build documentation

## Build Instructions

### macOS
```bash
./build_macos.sh
```

### Windows
```cmd
build_windows.bat
```

## Requirements

See your Python environment for required dependencies.
